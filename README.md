`git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm`

Download `.tmux.conf` to `~`.

Super key changed to `Ctrl+a`.



`tmux new -s session-name`

`Super + Ctrl+I` plugins

`SUPER + Ctrl+s` saves

`SUPER + Ctrl+r` restores

`SUPER + c` new tab

`SUPER + x` closes tab

`SUPER + &` closes session
